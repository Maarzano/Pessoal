atributos

Temos alguns atributos dentros das tags de HTML, como por exemplo a id, e a class:

  • id: dá um id único para identificação do elemento, para utilização do for com label por exemplo...
  • class: o class pode se repetir, ele serve principalmente para se colocar css em conjuntos específicos...
  • style: você coloca estilo scc diretamente no elemento (poucoi usado e não é recomendado).
  • title: o title é usado para dar um nome ao passar o mouse por cima do elemento.
  • específicos: alguns elementos possuem atributos específicos, como links, forms etc.

texto e formatação

Tag <strong> ✅

Usada para indicar importância no texto, sendo semântica e melhor para acessibilidade.

Tag <b> ❌

Usada para negrito, mas sem significado semântico. Prefira <strong> ou CSS.

Tag <em> ✅

Usada para dar ênfase ao texto, sendo interpretada corretamente por leitores de tela.

Tag <i> ❌

Usada para itálico, mas sem significado semântico. Prefira <em> ou CSS.

Tag <u> ❌

Usada para sublinhado, mas pode confundir com links. Use CSS.

Tag <s> ❌

Texto riscado. Pode ser feito com text-decoration: line-through; no CSS.

Tag <mark> ✅

Destaca o texto com um fundo amarelo, útil para marcações.

Tag <small> ✅

Usada para texto de aviso ou detalhes menores. Exemplo: Termos e condições aplicáveis.

Tag <abbr> ✅

Define abreviações. Exemplo: HTML.

Tag <q> ✅

Para citações curtas: O importante não é vencer todos os dias, mas lutar sempre.

Tag <blockquote> ✅

Para citações longas:

"O sucesso nasce do querer, da determinação e persistência em se chegar a um objetivo."

Tag <code> ✅

Para código fonte: console.log("Olá, mundo!");

Tag <pre> ✅

Para manter a formatação original do código:

                if (x > 10) {
                    console.log("Maior que 10");
                }

Tag <br> ✅

Quebra de linha manual: Primeiro texto.

Tag <span> ✅

Guia de Tags de Mídia no HTML

1. Imagem - Tag <img>

Usada para exibir imagens.

<img src="imagem.jpg" alt="Descrição da imagem" width="300">

2. Vídeo - Tag <video>

Usada para exibir vídeos.

<video src="video.mp4" controls></video>

3. Áudio - Tag <audio>

Usada para tocar arquivos de áudio.

<audio src="musica.mp3" controls></audio>

4. Incorporação de Conteúdo - Tag <iframe>

Usada para incorporar páginas externas, como vídeos do YouTube.

<iframe src="https://www.youtube.com/embed/VIDEO_ID" width="560" height="315"></iframe>

5. Gráficos Vetoriais - Tag <svg>

Usada para gráficos escaláveis.

                <svg width="100" height="100">
                <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red"/>
                </svg>

6. Canvas - Tag <canvas>

Usada para criar gráficos e desenhos com JavaScript.

                <canvas id="meuCanvas" width="200" height="100"></canvas>
                <script>
                    var c = document.getElementById("meuCanvas");
                    var ctx = c.getContext("2d");
                    ctx.fillStyle = "blue";
                    ctx.fillRect(10, 10, 150, 80);
                </script>

Tags de Tabela no HTML

1. Tag <table>

Define uma tabela em HTML.

2. Tag <tr>

Representa uma linha na tabela.

3. Tag <th>

Define um cabeçalho de coluna.

4. Tag <td>

Define uma célula de dados.

5. Atributo colspan

Faz uma célula ocupar múltiplas colunas.

6. Atributo rowspan

Faz uma célula ocupar múltiplas linhas.

7. Tag <thead>

Agrupa cabeçalhos de uma tabela.

8. Tag <tbody>

Agrupa o conteúdo principal da tabela.

9. Tag <tfoot>

Agrupa o rodapé da tabela.

10. Tag <caption>

Define um título para a tabela.

11. Tag <colgroup>

Define um grupo de colunas para estilização.

12. Tag <col>

Especifica propriedades para colunas dentro do <colgroup>.

13. Atributo scope

Define o escopo de um cabeçalho (<th>).
